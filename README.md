# React quiz test app

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Start instructions

```
npm install
npx playwright install
npm start
```

## To run tests
```
npm run test:e2e
```

## To run tests in UI mode
```
npm run test:e2e:debug
```
